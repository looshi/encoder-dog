About

Free online mp3 encoder. Edit tags and save album art from image file!

This page uses the ffmpeg Web Assembly library which allows ffmpeg to run in the browser. This program runs entirely on your computer within the browser, no data is sent to a server, no data is stored, no data is shared. Free to use!

  • ffmpeg: https://ffmpeg.org
  • wasm: https://webassembly.org
  • ffmpegwasm: https://ffmpegwasm.netlify.app
  • source code for this page: https://github.com/looshi/encoder-dog

Mp3 files are converted to constant bitrate of 320k.

To view ffmpeg output: right click the browser screen and select "inspect" and the click on the "console" tab.

Tested in Chrome on linux and windows. Other browsers may not work.

Convert music files ( wav , aiff, flac, etc. ) to mp3 and add tags right here in the browser!
